pre_download
============

利用python首先抓取出一个域名下的所有连接，然后使用代理的方式，预先访问一遍

使用方法：
--------

python pre_download.py  your_domain  proxy_ip

example  :

python pre_download.py www.qq.com  192.168.1.2
